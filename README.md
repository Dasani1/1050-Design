# Our 1050 Project! The Light Sensor Box
This is the code to our light sensor box that we created over the course of months and months of hard work.

# Languages:
C++ (Arduino) and Python (Script)

# Code Libraries:
Arduino: Wire.h, Liquid Crystal I2C, ArduinoJson,SPI.h,SD.h
Python: Pyinstaller, Matplotlib, json

# How to use:
This code assumes it's already built and the arduino code is fully loaded

1. Copy and Paste jsonreader.exe and jsonconvert.exe into your microsd
2. Run the arduino until a data.txt file is created 
3. Open the microsd on your computer and run the jsonconvert.exe file to create a Json file
4. Then run the jsonreader.exe to use the json data and paste it onto the graph (all you need to do is click)
All complete!
